# Fortress of Zorlac

A modern browser reimagining of the 1982 ZX81 shooter.

## Run

Open `index.html` in a modern desktop or mobile browser.

## Controls

- `WASD` or arrow keys: move
- Left click or `Space`: fire straight ahead
- `P`: pause or resume
- `Full Screen`: expand the play area

## Features

- Three nested rotating fortress walls with fine-grained damage
- Persistent local leaderboard stored in `localStorage`
- Start-level selection with higher-level score bonuses
- Fullscreen launch, touch controls, and generated synth SFX
